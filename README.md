一、项目介绍
---
基于SSM框架实现的体育器材预约系统包含两种角色：管理员、用户,系统分为前台和后台两大模块，主要功能如下。

学生端主要功能有注册、登录、个人信息管理、查看我的预约信息、归还器材等功能
### 
【老师】端需要实现的主要功能有登录、归还器材、个人信息管理、我的预约信息。
### 
【管理员】端主要功能有登录、用户管理、体育器材类别管理、器材管理、预约管理、数据统计、以及公告管理、个人信息管理等功能。
